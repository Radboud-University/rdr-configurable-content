Use your own username and password as a Radboud University or UMC employee to log into the Data Access application.

The username starts with the character u, s, e or z, followed by a number, and is appended with @ru.nl (for u-, s-, e-numbers) or @umcn.nl (for z-numbers).
For example: u100000@ru.nl or z1000000@umcn.nl.

Please visit the following page, if you need more information on the available Data Access applications, and how to use them.

You can still use the one time password option available below. Note that after each login a new password will be generated.
