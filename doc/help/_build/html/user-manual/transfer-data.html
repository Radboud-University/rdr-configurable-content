Transfer data

To transfer data, you should use a HTTP-based file transfer called WebDAV. To use this, you need a username and password.

Only RU and RadboudUMC employees can use the credentials provided by their employer (u-, s-, e- or z-number and its password) to use WebDAV file transfer.

  • Data access account

Below are instructions for how to use the recommended tools for transferring data to and from the repository. Depending on your operating system, please select one of the options.

  • Mount network drive (Windows and MacOSX)
  • Cyberduck (Windows and MacOSX)
  • Thunar (Linux - GUI)
  • Cadaver (Linux - command line)

After up- or downloading your files, make sure the transfer was completed accurately.
